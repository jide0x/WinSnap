Security Policy
===============

Reporting a Vulnerability
-------------------------

If you believe you have found a security issue in WinSnap, please do not open a public issue.

Instead, email the maintainers privately with:
- A description of the issue
- Steps to reproduce
- Potential impact

We will acknowledge receipt, investigate, and provide guidance.

Safe Disclosure
---------------
- Please allow a reasonable time for a fix to be developed and released before public disclosure.
